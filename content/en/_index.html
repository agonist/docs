What is Genji?

Genji is an embedded database written in Go that aims to simplify dealing with data in the modern world. It combines the power of SQL with the versatility of documents to provide a maximum of flexibility with no compromise.

{{< /blocks/lead >}} {{< blocks/section >}} {{% blocks/feature icon="fas fa-database" title="The power of SQL with documents" %}} Run powerful queries on rich documents with an expressive SQL syntax. {{% /blocks/feature %}} {{% blocks/feature icon="fa-border-style" title="Flexible schemas" %}} Create tables with strict schemas, partial schemas, or no schemas at all to control your data the way you want it. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-rocket" title="Feature rich" %}} Fully serializable transactions, in-memory mode, memory usage control, and more. {{% /blocks/feature %}} {{< /blocks/section >}}
